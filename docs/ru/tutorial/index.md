# Туториал — Руководство пользователя

Этот туториал проведёт тебя через все возможности **django-pydantic-client** — от установки до продвинутого использования.

Каждый раздел строится на предыдущем. Можно читать по порядку или перейти сразу к нужной теме.

## Разделы

| Раздел | Что узнаешь |
|--------|------------|
| [Первые шаги](first-steps.md) | Установка, настройка, первая валидированная вьюха |
| [Схема запроса](request-schema.md) | Как работает `RequestModel` изнутри |
| [Источники данных](request-data.md) | Откуда берутся данные: JSON, форма, GET, URL-параметры |
| [Ошибки валидации](validation.md) | 422-ответы, формат ошибок, кастомная обработка |
| [Параметры URL](url-params.md) | Передача path-параметров в схему |
| [Возможности Pydantic](pydantic-features.md) | `Field()`, валидаторы, `model_config`, вычисляемые поля |

## Что нужно знать заранее

- Базовый Django: вьюхи, URLs, `HttpRequest`
- Базовый Pydantic: `BaseModel`, аннотации типов

Знания Django REST Framework не требуются.
